# Credit Card Fraud Detection

## Project Overview

The Credit Card Fraud Detection project is a machine learning application that identifies fraudulent credit card transactions based on transaction details. The model analyzes transaction patterns and classifies transactions as either genuine or fraudulent.

## Objective

The objective of this project is to build a classification model that can accurately detect fraudulent credit card transactions using machine learning techniques.

## Dataset

The dataset contains transaction-related information with the following features:

* Amount
* Time
* Fraud Status

The target variable indicates whether a transaction is genuine (0) or fraudulent (1).

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Machine Learning Algorithm

* Logistic Regression

## Project Workflow

1. Load the transaction dataset.
2. Preprocess and prepare the data.
3. Split the dataset into training and testing sets.
4. Train the Logistic Regression model.
5. Predict fraudulent and genuine transactions.
6. Evaluate the model using classification metrics.

## Results

The model successfully classified transactions as fraudulent or genuine and provided performance metrics such as precision, recall, and F1-score.

## Learning Outcomes

* Understanding classification-based machine learning models.
* Data preprocessing and train-test splitting.
* Fraud detection using Logistic Regression.
* Model evaluation using classification metrics.
